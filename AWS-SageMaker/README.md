# Two-Week Intensive AWS SageMaker Learning Plan

This learning plan is designed to help you master AWS SageMaker, with the ultimate goal of clearing the AWS Machine Learning Specialization.

- [Introduction to Amazon SageMaker](https://www.youtube.com/watch?v=U7C_67vB6z4)

### Day 5: Understanding the SageMaker Model Lifecycle
